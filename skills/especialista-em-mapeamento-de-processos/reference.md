# Expert in Process Mapping — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **AS-IS mapping** — current state.
- **SIPOC** — suppliers-inputs-process-outputs-customers.
- **Value stream mapping** — flow + delays.

## Tooling / Stack
- **Discovery:** interviews, gemba walks, observation.
- **Tools:** SIPOC, VSM, swimlane maps.
- **Process mining:** event-log discovery.

## Detailed checklists
- [ ] Process scoped (SIPOC)
- [ ] Real performers interviewed
- [ ] Waste/delays/handoffs visible
- [ ] Validated by doers

## Common pitfalls
- **Idealized mapping.**
- **Desk-based guessing.**
- **No quantification.**

## Further patterns
- **SIPOC then detailed map.**
- **Value stream mapping.**
