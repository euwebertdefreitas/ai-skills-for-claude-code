# Expert in Executive Decision Making — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **One-way vs two-way doors** — reversibility.
- **Opportunity cost** — the road not taken.
- **Second-order thinking** — and then what?

## Tooling / Stack
- **Frameworks:** decision matrices, scenarios.
- **Risk:** pre-mortems, red teams.
- **Bets:** expected value, real options.

## Detailed checklists
- [ ] Reversibility classified
- [ ] Criteria set before options
- [ ] Trade-offs/opportunity cost weighed
- [ ] Clear commit and communicate

## Common pitfalls
- **Analysis paralysis** on reversible calls.
- **HiPPO bias** — highest-paid opinion.
- **Ignoring second-order effects.**

## Further patterns
- **Disagree and commit.**
- **Pre-mortem before big bets.**
