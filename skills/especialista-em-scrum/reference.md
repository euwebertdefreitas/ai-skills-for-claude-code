# Expert in Scrum — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Three pillars** — transparency, inspection, adaptation.
- **Roles** — PO, Scrum Master, Developers.
- **Events/artifacts** — sprint, backlog, increment.

## Tooling / Stack
- **Boards:** Jira, Azure Boards.
- **Events:** planning, daily, review, retro.
- **Artifacts:** product/sprint backlog, increment.

## Detailed checklists
- [ ] Roles clear and respected
- [ ] Events purposeful and timeboxed
- [ ] DoD enforced
- [ ] Retro actions followed up

## Common pitfalls
- **Mechanical Scrum** — keep the why.
- **Overcommitted sprints** — pace sustainably.
- **No DoD** — quality drifts.

## Further patterns
- **Backlog refinement cadence.**
- **Sprint goal focus.**
