# Expert in Kanban — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **WIP limits** — pull, not push.
- **Lead vs cycle time** — flow measures.
- **Cumulative flow diagram (CFD).**

## Tooling / Stack
- **Boards:** physical/digital Kanban boards.
- **Metrics:** CFD, cycle-time scatterplots.
- **Cadences:** replenishment, delivery reviews.

## Detailed checklists
- [ ] Workflow visualized
- [ ] WIP limits set/enforced
- [ ] Flow metrics tracked
- [ ] Bottlenecks addressed

## Common pitfalls
- **No WIP limits.**
- **Starting over finishing.**
- **Ignoring blockers.**

## Further patterns
- **Pull system with WIP limits.**
- **Classes of service.**
