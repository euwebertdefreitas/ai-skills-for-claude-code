# Expert in Software Quality — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Cost of quality** — prevention vs failure cost.
- **Shift-left** — find defects earlier and cheaper.
- **Technical debt** — deferred quality with interest.

## Tooling / Stack
- **Quality gates:** SonarQube, coverage thresholds.
- **Metrics:** DORA, defect tracking.
- **Standards:** Definition of Done, coding standards.

## Detailed checklists
- [ ] DoD defined and enforced
- [ ] Quality gates in CI
- [ ] Meaningful metrics tracked
- [ ] Debt visible and managed

## Common pitfalls
- **Vanity metrics** — measure outcomes.
- **Gatekeeper QA** — distribute ownership.
- **Debt blindness** — track it like work.

## Further patterns
- **Shift-left quality.**
- **Quality gates in the pipeline.**
