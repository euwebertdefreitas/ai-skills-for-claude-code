# Expert in Product Management — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Discovery vs delivery** — learn vs build.
- **Outcome over output** — value focus.
- **Prioritization frameworks** — RICE, Kano, value/effort.

## Tooling / Stack
- **Discovery:** interviews, opportunity solution trees.
- **Prioritization:** RICE, MoSCoW.
- **Analytics:** funnels, retention, A/B tests.

## Detailed checklists
- [ ] Problem validated with users
- [ ] Success metrics defined
- [ ] Backlog impact-prioritized
- [ ] Roadmap outcome-oriented

## Common pitfalls
- **Solution-first** — start with the problem.
- **Stakeholder roadmaps** — anchor on outcomes.
- **Vanity metrics** — track real value.
