# Expert in Project Management — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Triple constraint** — scope/time/cost balance.
- **WBS** — work breakdown structure.
- **Risk register** — proactive risk management.

## Tooling / Stack
- **Planning:** Gantt, WBS, critical path.
- **Tracking:** status reports, EVM.
- **Frameworks:** PMBOK, PRINCE2.

## Detailed checklists
- [ ] Scope/success defined
- [ ] Schedule and resources baselined
- [ ] Risks registered with responses
- [ ] Change control in place

## Common pitfalls
- **Scope creep** — enforce change control.
- **Optimistic estimates** — buffer and validate.
- **Stakeholder silence** — over-communicate.

## Further patterns
- **Critical path / critical chain.**
- **Stage-gate delivery.**
