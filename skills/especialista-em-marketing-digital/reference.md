# Expert in Digital Marketing — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **SEO vs SEM** — organic vs paid intent.
- **Attribution** — crediting conversions.
- **ROAS/CPA** — paid efficiency metrics.

## Tooling / Stack
- **Analytics:** GA4, Tag Manager.
- **Ads:** Google/Meta Ads.
- **Automation:** CRM/email/retargeting.

## Detailed checklists
- [ ] Conversions tracked + UTMs
- [ ] Channel matches intent
- [ ] A/B tests running
- [ ] ROAS/CPA monitored

## Common pitfalls
- **Traffic over conversion.**
- **No attribution.**
- **Channel sprawl.**

## Further patterns
- **Full-funnel + retargeting.**
- **SEO+SEM combination.**
