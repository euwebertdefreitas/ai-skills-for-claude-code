# Expert in Service Desk / ITSM — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Incident vs problem vs change** — distinct ITIL processes.
- **SLA/OLA** — service-level agreements.
- **Service catalog** — offered services.

## Tooling / Stack
- **ITSM:** ServiceNow, Jira Service Management.
- **Frameworks:** ITIL 4.
- **Metrics:** SLA dashboards, CSAT.

## Detailed checklists
- [ ] Incidents prioritized by impact/urgency
- [ ] Problem management active
- [ ] Change risk-managed
- [ ] SLAs measured/met

## Common pitfalls
- **No root-cause (problem) mgmt.**
- **Uncontrolled changes.**
- **Unmeasured SLAs.**

## Further patterns
- **ITIL incident/problem/change.**
- **Shift-left / self-service.**
