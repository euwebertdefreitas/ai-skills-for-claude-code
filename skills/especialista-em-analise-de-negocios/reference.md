# Expert in Business Analysis — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Current vs future state / gap analysis.**
- **Stakeholder analysis** — interest/influence.
- **Value/benefit realization** — measurable outcomes.

## Tooling / Stack
- **Analysis:** SWOT, root cause, value chain.
- **Modeling:** process maps, capability maps.
- **Frameworks:** BABOK techniques.

## Detailed checklists
- [ ] Problem and value framed
- [ ] Stakeholders mapped
- [ ] Gap (current→future) defined
- [ ] Benefits quantified

## Common pitfalls
- **Solution bias** — frame the problem first.
- **Stakeholder blind spots** — map fully.
- **Unmeasured value** — define metrics.

## Further patterns
- **Gap analysis (current→future).**
- **Value stream / capability mapping.**
