# Expert in Design Thinking — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Five stages** — empathize, define, ideate, prototype, test.
- **Divergent/convergent thinking** — expand then focus.
- **HMW statements** — actionable problem frames.

## Tooling / Stack
- **Research:** interviews, empathy maps, journey maps.
- **Ideation:** brainstorming, crazy-8s.
- **Prototyping:** paper, clickable mockups.

## Detailed checklists
- [ ] User needs understood (empathy)
- [ ] Problem reframed (define)
- [ ] Many ideas generated
- [ ] Prototype tested with users

## Common pitfalls
- **Solutioning early** — empathize first.
- **Idea fixation** — diverge widely.
- **Over-polished prototypes** — keep them cheap.
