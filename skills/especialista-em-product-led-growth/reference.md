## Key concepts
- **Time-to-value (TTV)** — speed to first value.
- **PQL** — usage-based qualified lead.
- **Growth loops** — self-reinforcing acquisition.

## Tooling / Stack
- **Analytics:** product analytics (Amplitude/Mixpanel).
- **Onboarding:** in-app guides, checklists.
- **Experimentation:** A/B testing.

## Detailed checklists
- [ ] Activation moment defined/measured
- [ ] Onboarding delivers value fast
- [ ] PQL signals identified
- [ ] Growth loop in place

## Common pitfalls
- **Early paywall** — show value first.
- **Feature tours** — focus on value, not features.
- **Signup-only metrics** — track activation.

## Further patterns
- **Freemium / free-trial funnels.**
- **Viral / network growth loops.**
