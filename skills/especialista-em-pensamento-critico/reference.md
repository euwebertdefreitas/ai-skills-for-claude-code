# Expert in Critical Thinking — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Validity vs soundness** — logic vs true premises.
- **Common fallacies** — ad hominem, false dilemma, etc.
- **Burden of proof** — who must support the claim.

## Tooling / Stack
- **Frameworks:** argument mapping, Toulmin model.
- **Checklists:** fallacy lists, assumption audits.
- **Methods:** Socratic questioning.

## Detailed checklists
- [ ] Premises and conclusion identified
- [ ] Inference validity checked
- [ ] Fallacies named
- [ ] Alternatives considered

## Common pitfalls
- **Strawmanning** — represent fairly.
- **Confirmation bias** — seek disconfirmation.
- **Equivocation** — define terms.

## Further patterns
- **Steelman then critique.**
- **Toulmin argument analysis.**

## Further reading
- Thinking, Fast and Slow (Kahneman).
- A Rulebook for Arguments (Weston).
