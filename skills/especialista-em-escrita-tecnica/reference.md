# Expert in Technical Writing — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **BLUF** — bottom line up front.
- **Plain language** — clarity over formality.
- **Audience adaptation** — depth to fit.

## Tooling / Stack
- **Editing:** Hemingway, readability checks.
- **Style:** plain-language guides.
- **Structure:** outlines, top-down.

## Detailed checklists
- [ ] Key point first
- [ ] Active voice, short sentences
- [ ] Jargon defined/avoided
- [ ] Audience-appropriate depth

## Common pitfalls
- **Buried lede.**
- **Passive/wordy prose.**
- **Audience mismatch.**

## Further patterns
- **BLUF structure.**
- **Inverted pyramid.**

## Further reading
- The Elements of Style (Strunk & White).
- On Writing Well (Zinsser).
