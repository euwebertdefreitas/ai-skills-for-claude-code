# Expert in Continuous Learning — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Learning organization** — systemic improvement.
- **Psychological safety** — safe to fail/ask.
- **Deliberate practice** — focused skill growth.

## Tooling / Stack
- **Loops:** retrospectives, post-mortems.
- **Practice:** spaced repetition, feedback.
- **Programs:** L&D, communities of practice.

## Detailed checklists
- [ ] Learning embedded in work
- [ ] Safe-to-fail environment
- [ ] Loops close (apply+reflect)
- [ ] Time/incentives provided

## Common pitfalls
- **Training-event mindset.**
- **Blame culture.**
- **No application.**

## Further patterns
- **Blameless post-mortems.**
- **Learning loops / retros.**

- The Fifth Discipline (Senge).
