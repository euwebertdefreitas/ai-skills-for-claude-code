# Expert in Helpdesk Support — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Tiered support** — L1/L2/L3 escalation.
- **Troubleshooting method** — reproduce, isolate, fix.
- **Ticket lifecycle** — open→resolve→close.

## Tooling / Stack
- **Ticketing:** Zendesk, Freshdesk, Jira SM.
- **Knowledge:** KB articles, FAQs.
- **Remote:** remote-access/diagnostic tools.

## Detailed checklists
- [ ] Symptoms gathered/reproduced
- [ ] KB checked
- [ ] User confirms resolution
- [ ] Solution documented

## Common pitfalls
- **Guess-and-check fixes.**
- **Premature closure.**
- **No documentation.**

## Further patterns
- **Reproduce-isolate-fix.**
- **Knowledge-centered support (KCS).**
