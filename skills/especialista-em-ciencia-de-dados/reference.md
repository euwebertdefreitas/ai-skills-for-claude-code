# Expert in Data Science — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Bias-variance trade-off** — under/overfitting balance.
- **Cross-validation** — robust performance estimation.
- **Feature leakage** — info that wouldn't exist at predict time.

## Tooling / Stack
- **Python:** pandas, NumPy, scikit-learn, statsmodels.
- **Viz:** matplotlib, seaborn, Plotly.
- **Notebooks:** Jupyter, with versioned outputs.

## Detailed checklists
- [ ] EDA done; data quality understood
- [ ] Baseline established
- [ ] Validation leak-free
- [ ] Uncertainty and assumptions reported

## Common pitfalls
- **Simpson's paradox** — check subgroup effects.
- **Imbalanced metrics** — use precision/recall/AUC.
- **P-hacking** — pre-register hypotheses.

## Further patterns
- **CRISP-DM** — structured DS lifecycle.
- **Baseline → iterate** modeling discipline.
