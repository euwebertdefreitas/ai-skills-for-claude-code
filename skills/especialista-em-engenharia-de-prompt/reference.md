# Expert in Prompt Engineering — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Few-shot / in-context learning** — examples in the prompt.
- **Chain-of-thought** — explicit reasoning steps.
- **System vs user prompt** — role and stable instructions.

## Tooling / Stack
- **Models:** Claude, GPT, open models.
- **Eval:** golden sets, LLM-as-judge.
- **Structure:** delimiters, schemas, output formats.

## Detailed checklists
- [ ] Role/task/constraints/format explicit
- [ ] Examples for format and edge cases
- [ ] Reasoning enabled where useful
- [ ] Evaluated on a set, not one case

## Common pitfalls
- **Ambiguity** — over-specify format.
- **Prompt bloat** — keep it focused.
- **Overfitting** — test broadly.

## Further patterns
- **Few-shot + structured output.**
- **Decompose-and-chain prompts.**
