# Expert in Hospital Management — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Patient flow** — throughput from admit to discharge.
- **Length of stay / occupancy** — capacity metrics.
- **Revenue cycle** — billing to collection.

## Tooling / Stack
- **Systems:** HIS/EHR, bed management.
- **Ops:** capacity/flow analytics, scheduling.
- **Finance:** revenue cycle management.

## Detailed checklists
- [ ] Patient flow optimized
- [ ] Capacity (beds/OR/ED) managed
- [ ] Staffing matched to demand
- [ ] Supply/revenue cycle controlled

## Common pitfalls
- **ED/discharge bottlenecks.**
- **Unsafe understaffing.**
- **Poor discharge planning.**

## Further patterns
- **Patient flow / lean healthcare.**
- **Capacity command center.**
