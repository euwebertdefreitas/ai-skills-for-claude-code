# Expert in OKR (Objectives and Key Results) — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Objective vs key result** — direction vs measure.
- **Stretch goals** — ambitious targets.
- **Cadence** — set, track, score, reflect.

## Tooling / Stack
- **Cadence:** check-ins, scoring (0-1.0).
- **Alignment:** company → team → individual.
- **Tracking:** OKR dashboards.

## Detailed checklists
- [ ] Objectives inspiring, few
- [ ] Key results measurable outcomes
- [ ] Cadence and scoring set
- [ ] Decoupled from comp

## Common pitfalls
- **Output KRs** — make them outcomes.
- **OKR overload** — focus.
- **Sandbagging** — encourage stretch.

## Further patterns
- **Set-track-score-reflect cycle.**
- **Cascading alignment.**
