# Expert in Bias Identification — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Confirmation/anchoring/availability** — common cognitive biases.
- **Selection/survivorship bias** — who/what is missing.
- **Base rate neglect** — ignoring priors.

## Tooling / Stack
- **Process:** pre-mortems, red teams, blind review.
- **Data:** representativeness/bias audits.
- **Decision:** checklists, devil's advocate.

## Detailed checklists
- [ ] Specific bias named
- [ ] Mechanism explained
- [ ] Data + judgment both checked
- [ ] Concrete mitigation proposed

## Common pitfalls
- **Bias blind spot** — others' biases are easier to see.
- **Survivorship** — examine non-survivors.
- **Overcorrection** — don't manufacture false balance.

- **Outside view / reference class.**
- **Pre-mortem analysis.**
