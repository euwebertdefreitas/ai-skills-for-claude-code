# Expert in User Training and Enablement — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Adult learning (andragogy)** — relevance + practice.
- **Task-based training** — real workflows.
- **Adoption vs awareness.**

## Tooling / Stack
- **Delivery:** workshops, e-learning, walkthroughs.
- **Aids:** quick-reference guides, in-app help.
- **Measurement:** proficiency checks, adoption metrics.

## Detailed checklists
- [ ] Role/task-based design
- [ ] Hands-on practice
- [ ] Job aids provided
- [ ] Proficiency/adoption measured

## Common pitfalls
- **Feature tours.**
- **Lecture-only.**
- **No reinforcement.**

## Further patterns
- **Learn-by-doing labs.**
- **Train-the-trainer.**
