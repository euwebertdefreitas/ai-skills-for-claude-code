# Expert in Fact-Checking — Reference

Detailed companion to SKILL.md. Load on demand when a task needs depth beyond the core workflow.

## Key concepts
- **Primary vs secondary source** — origin vs report.
- **Corroboration** — independent confirmation.
- **Provenance** — where a claim originated.

## Tooling / Stack
- **Search:** reverse image search, archives.
- **Sources:** primary datasets, official records.
- **Method:** lateral reading, SIFT.

## Detailed checklists
- [ ] Primary source located
- [ ] Multiple independent corroborations
- [ ] Uncertainty flagged
- [ ] Citations exact

## Common pitfalls
- **Circular reporting** — sources citing each other.
- **Outdated data** — check recency.
- **Out-of-context quotes** — verify full context.

## Further patterns
- **SIFT (Stop, Investigate, Find, Trace).**
- **Lateral reading.**
